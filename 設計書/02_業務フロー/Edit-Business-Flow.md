```uml
@startuml
title: 会員情報編集
opt 会員情報編集
        ユーザー ->Webサーバー :更新する情報及び現在のパスワード入力\n(変更ない場合はそのまま)し、確定ボタンクリック
        Webサーバー -> DBサーバー : 現在のメールアドレス[セッションデータ]とパスワード参照
        alt 一致
                alt 新規パスワードが8文字以上で、英小文字、\n英大文字、数字を各1文字以上含むorパスワード更新無し
                       Webサーバー -> DBサーバー : 更新許可
                       DBサーバー -> DBサーバー : 更新処理
                else パスワードが適切(上記条件)ではない
                        Webサーバー -> ユーザー : パスワードが適切でないことを表示
                end
        else 一致しない
                Webサーバー -> ユーザー : パスワードが一致しないことを表示
        end
end
@enduml
```
